Learning outcomes
By the end of this lecture, you will be able to describe mechanisms for cell-to-
cell communication in a complex multicellular organism (chemical & electrical).
This includes being able to:
• Describe the different types of chemical signals between cells using
examples.
• Discuss how action potentials are generated and propagated.
• Compare and contrast chemical and electrical signalling between cells.
