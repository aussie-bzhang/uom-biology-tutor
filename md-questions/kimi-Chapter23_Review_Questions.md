## Question 1: Test Cross Experiment

**(a) Experimental Design**

**Parental genotypes and phenotypes:** Unknown parent: P/? (purple phenotype). Tester parent: p/p (white phenotype, homozygous recessive).

**Cross setup:** P/? x p/p

Punnett Squares for both scenarios:

If unknown is P/P (homozygous): P/P x p/p --> All offspring = P/p (purple). Ratio = 100% purple.

If unknown is P/p (heterozygous): P/p x p/p --> 1/2 P/p (purple) : 1/2 p/p (white). Ratio = 1:1.

**Interpretation:** If ANY white offspring appear, the unknown parent MUST be heterozygous (P/p). If all offspring are purple (with a sufficiently large sample, e.g., 20+ offspring), the parent is likely homozygous dominant (P/P).

**(b) Binomial Probability Calculation**

Using the binomial formula: P(k) = C(n,k) x p^k x q^(n-k)

Where: n = 4, k = 3, p = 0.5 (probability of purple), q = 0.5 (probability of white)

C(4,3) = 4! / (3! x 1!) = 4

**P(3 purple) = 4 x (0.5)^3 x (0.5)^1 = 4 x 0.125 x 0.5 = 0.25 or 25%**

## Question 2: X-Linked Inheritance

**(a) Woman's Genotype**

The woman's father had hemophilia (genotype: X^h Y). A father passes his X chromosome to ALL daughters. Therefore, the woman MUST have inherited X^h from her father. Her mother is normal, so she contributed X^H.

**Woman's genotype:** X^H X^h (carrier, heterozygous)

**(b) Punnett Square and Probabilities**

Cross: X^H X^h (carrier female) x X^H Y (normal male)

Punnett square results: 1/4 X^H X^H (normal female), 1/4 X^H X^h (carrier female), 1/4 X^H Y (normal male), 1/4 X^h Y (hemophilic male).

**(i) P(son with hemophilia) =** P(X^h Y) = 1/4 or 25%

**(ii) P(carrier daughter) =** P(X^H X^h) = 1/4 or 25%

**(c) Extended Probability**

P(affected child) = P(X^h Y) = 1/4 (only sons can be affected).

P(unaffected child) = 1 - 1/4 = 3/4

**P(none of 3 children affected) = (3/4)^3 = 27/64 = 42.19%**

## Question 3: Pedigree Analysis

**(a) Mode of Inheritance**

**Most likely: Autosomal recessive.**

Key evidence: (1) Affected individuals (II-1, III-1) have unaffected parents, indicating the trait skips a generation. (2) Both sexes are affected. (3) I-1 and I-2 are both carriers.

Inconsistencies for alternative modes:

• Autosomal dominant: II-1 is affected but both I-1 and I-2 are unaffected --> INCONSISTENT (affected children must have at least one affected parent).

• X-linked dominant: II-1 is an affected male with unaffected father I-1. For X-linked dominant, an affected son must receive the affected X from his mother, making the mother affected. But I-2 is unaffected --> INCONSISTENT.

• X-linked recessive: II-1 (X^a Y) requires mother I-2 to be at least a carrier (X^A X^a). This is possible. However, III-1 (affected male) requires II-2 to be a carrier. II-2 is unaffected but has an affected brother, so she has 2/3 chance of being a carrier (if autosomal) or 1/2 chance (if X-linked, from mother I-2). Both are possible, but autosomal recessive is more parsimonious given equal male/female affected ratio.

**(b) Genotypes (using A = normal, a = affected)**

**I-1:** Aa (carrier, unaffected male)

**I-2:** Aa (carrier, unaffected female)

**II-2:** Aa (must be carrier since her son III-1 is affected 'aa')

**III-2:** 1/3 AA or 2/3 Aa. Since her mother II-2 is Aa and father is unaffected with no family history (assumed AA), III-2 is: 1/2 AA : 1/2 Aa. Given she is unaffected, the ratio remains 1/2 AA : 1/2 Aa (or 1:1).

**(c) Probability for IV-1**

For IV-1 to be affected (aa): III-2 must be Aa AND pass 'a' allele, AND III-2's partner must also contribute 'a'.

P(III-2 = Aa) = 1/2 (from part b). P(III-2 passes 'a') = 1/2.

Assuming the trait is rare, III-2's partner is most likely AA (not a carrier). If partner is AA, P(affected) = 0.

If partner has unknown status but no family history: P(partner = Aa) = 2q where q is allele frequency (very small for rare traits).

With population carrier frequency ~1/50: P(IV-1 affected) = P(III-2=Aa) x P(passes a) x P(partner=Aa) x P(partner passes a) = (1/2) x (1/2) x (1/50) x (1/2) = 1/400 = 0.25%

(Note: If III-2's genotype is 2/3 Aa based on different parental assumptions, adjust accordingly: (2/3) x (1/2) x (1/50) x (1/2) = 1/300 = 0.33%)

## Question 4: Reciprocal Cross

**(a) Reciprocal Crosses**

**Cross A:** Crimson male (C/C) x Ivory female (c/c)

F1: All C/c (crimson) -- both males and females

F1 x F1 intercross: C/c x C/c --> F2: 3 crimson (C/C or C/c) : 1 ivory (c/c)

F2 ratio: 75% crimson, 25% ivory (same for both sexes)

**Cross B:** Ivory male (c/c) x Crimson female (C/C)

F1: All C/c (crimson) -- both males and females

F1 x F1 intercross: C/c x C/c --> F2: 3 crimson : 1 ivory

**(b) Interpretation**

Both reciprocal crosses produce IDENTICAL F1 and F2 results. This demonstrates that the trait is AUTOSOMAL -- the sex of the parent does not affect inheritance of the trait.

If the gene were X-linked: Cross A (crimson male X^C Y x ivory female X^c X^c) would give all crimson offspring in F1 (X^C X^c daughters and X^c Y sons). But Cross B (ivory male X^c Y x crimson female X^C X^C) would give X^C X^c daughters (crimson) and X^C Y sons (crimson) in F1 -- actually all crimson too. The difference would appear in F2: with X-linkage, the F2 from Cross A intercross would show crimson females, carrier females, crimson males, and ivory males (not a 3:1 overall ratio but sex-specific ratios).

**(c) Back Cross**

F1 crimson bird (genotype C/c, since it always is from pure lines) back-crossed to ivory tester (c/c):

Cross: C/c x c/c --> Offspring: 1/2 C/c (crimson) : 1/2 c/c (ivory).

A 1:1 ratio of crimson to ivory offspring confirms the F1 bird was heterozygous.

## Question 5: Comprehensive Probability

**(a) Dihybrid Cross**

Woman's mother was albino non-taster (ttaa). Therefore the woman MUST inherit 't' and 'a' from mother. Woman is a taster with normal pigmentation, so she inherited 'T' and 'A' from father.

**Woman's genotype:** TtAa (heterozygous for both)

**Man's genotype:** TtAa (heterozygous for both)

Cross: TtAa x TtAa. Using branch method:

P(taster T\_) = 3/4; P(non-taster tt) = 1/4

P(normal pigment A\_) = 3/4; P(albino aa) = 1/4

**(i) P(albino non-taster) =** P(ttaa) = (1/4) x (1/4) = 1/16 = 6.25%

**(ii) P(taster with normal pigment) =** P(T\_ A\_) = (3/4) x (3/4) = 9/16 = 56.25%

**(b) Conditional Probability**

P(Taster | Normal pigment) = P(Taster AND Normal pigment) / P(Normal pigment)

Numerator: P(T\_ A\_) = 9/16

Denominator: P(A\_) = 3/4 = 12/16

P(Taster | Normal pigment) = (9/16) / (12/16) = 9/12 = 3/4

Since the genes assort independently, tasting status is independent of pigmentation status. Knowing the child has normal pigment gives no information about tasting ability, so P(Taster | Normal) = P(Taster) = 3/4.

**(c) Experimental Verification**

To test for linkage, perform a TEST CROSS: cross a dihybrid individual (TtAa) with a double recessive tester (ttaa).

If genes assort independently: offspring ratio = 1:1:1:1 (taster-normal : taster-albino : non-taster-normal : non-taster-albino), approximately equal numbers of all four phenotypic classes.

If linked: parental phenotypes (taster-normal and non-taster-albino) will appear more frequently than recombinant phenotypes (taster-albino and non-taster-normal). The recombination frequency will be less than 50%. A chi-square test comparing observed to expected 1:1:1:1 ratio can confirm: if chi-square p-value < 0.05, reject independent assortment.
