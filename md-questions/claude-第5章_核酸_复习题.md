# 第一套 参考答案

### 一、问答题

1. DNA 四种碱基：腺嘌呤 (A)、胞嘧啶 (C)、鸟嘌呤 (G)、胸腺嘧啶 (T)。RNA 四种碱基：A、C、G、尿嘧啶 (U)。关键差异：RNA 用 U 代替 T；此外 DNA 含脱氧核糖、通常双链，RNA 含核糖、通常单链。
2. 核苷 (nucleoside) = 碱基 + 五碳糖 (核糖或脱氧核糖)；核苷酸 (nucleotide) = 核苷 + 磷酸基团 (即碱基 + 糖 + 磷酸)。换言之，核苷酸比核苷多了一个磷酸。
3. 磷酸二酯键是连接相邻核苷酸的共价键，连接一个核苷酸的 3' 羟基与下一个核苷酸 5' 端的磷酸，构成糖-磷酸骨架。它通过脱水缩合反应 (condensation reaction) 形成 (脱去一分子水)。
4. 中心法则的信息流向为 DNA → RNA → 蛋白质：DNA 通过转录 (transcription) 把信息传给 RNA，RNA 通过翻译 (translation) 把信息传给多肽。其中「RNA → 蛋白质」是不可逆的——信息绝不会从蛋白质回传到核酸。(DNA 还能通过复制 replication 自我复制。)
5. 编码链 (coding/sense strand) 的碱基序列与 mRNA 几乎相同 (只是 T 换成 U)；模板链 (template/antisense/non-coding strand) 是 RNA 聚合酶实际读取并据以合成 RNA 的那条链。因此 mRNA 序列与编码链最相似。

### 二、是非题

1. 对 (Yes)。两者都形成不分支的线性多核苷酸链。
2. 错 (No)。DNA 中是 A 与 T 配对、C 与 G 配对。
3. 对 (Yes)。G–C 之间有 3 个氢键，A–T 之间有 2 个氢键，故 G–C 更稳定。
4. 错 (No)。RNA 虽为单链，但分子内部的碱基可以互相配对，形成折叠结构 (如 tRNA)。
5. 对 (Yes)。AUG 编码甲硫氨酸，是翻译起始的第一个氨基酸。

# 第二套 参考答案

### 一、问答题

1. mRNA (信使 RNA)：携带从基因转录来的遗传信息，作为翻译的模板。tRNA (转运 RNA)：携带相应氨基酸到核糖体，通过反密码子识别 mRNA 上的密码子，把氨基酸装配成多肽。rRNA (核糖体 RNA)：核糖体 (蛋白质合成工厂) 的结构成分。三者共同参与由氨基酸合成蛋白质的过程。
2. 起始 (initiation)：RNA 聚合酶结合到启动子，DNA 解旋；延伸 (elongation)：聚合酶沿模板链移动，按碱基互补加入核糖核苷三磷酸，合成 RNA 链 (5'→3')，其后 DNA 重新复旋；终止 (termination)：到达终止位点，RNA 转录本释放，聚合酶脱离。催化酶为 RNA 聚合酶 (RNA polymerase)。
3. ORF 是一段连续、不重叠的密码子，从起始密码子 AUG 开始，到终止密码子 (UAA、UAG 或 UGA) 结束。起点由起始密码子决定，终点由终止密码子决定。
4. 因为读取起点不同会得到不同的密码子分组：在正向链上可从第 1、2、3 位起始 (+1、+2、+3)；互补链也能 5'→3' 方向读取，同样有三种起始 (−1、−2、−3)，合计 6 个阅读框。
5. tRNA 折叠成三叶草/L 形结构。反密码子 (anticodon) 的作用是通过碱基互补识别并结合 mRNA 上对应的密码子，从而保证正确的氨基酸被加入。氨基酸附着位点末端的固定序列是 CCA (3' 端)。

2. 对 (Yes)。大多数情况下一个基因编码一条多肽。
3. 错 (No)。转录发生在 (真核细胞) 细胞核内，翻译发生在核糖体上 (位置说反了)。
4. 对 (Yes)。乙肝病毒约一半基因组含 4 个重叠 ORF (S、P、C、X)。
5. 错 (No)。控制转录起止的信号与控制翻译起止的信号不同。

# 第三套 参考答案

### 一、问答题

1. 差异对照：链数——DNA 双链 / RNA 单链；糖——DNA 脱氧核糖 (2' 无 −OH) / RNA 核糖 (2' 有 −OH)；碱基——DNA 为 A、C、G、T / RNA 为 A、C、G、U。
2. 互补链：3'-CTAATGT-5' (即与 5'-GATTACA-3' 配对)。规则：A 与 T 配对 (2 个氢键)、G 与 C 配对 (3 个氢键)；两条链反平行。(若按 5'→3' 书写互补链，则为 5'-TGTAATC-3'。)
3. 核酸链的两端化学性质不同：5' 端带有 (游离) 磷酸基团，3' 端带有游离的 3'-羟基。由于糖-磷酸骨架方向固定 (5'→3')，链具有方向性。新核苷酸通常加在链的 3' 端 (即链以 5'→3' 方向延伸)。
4. 基因是编码一条多肽(蛋白质)的一段 DNA 序列。染色体由长链 DNA 高度压缩盘绕而成，许多基因沿 DNA 分布在染色体上。编码区 (coding region) 被转录并翻译为蛋白质；非编码区 (non-coding region) 包括被转录但不翻译的部分，以及既不转录也不翻译的部分 (如启动子 promoter)。
5. 完整过程：DNA 中的基因先被转录 (transcription) 成前体 RNA (pre-mRNA) → 经过加工 (processing) 成为成熟 mRNA → mRNA 离开细胞核到核糖体 → 在翻译 (translation) 中，tRNA 按反密码子-密码子配对把对应氨基酸依次加上，从起始密码子 AUG 开始、到终止密码子结束 → 形成多肽链 (从 N 端到 C 端)。

### 四、转录与翻译应用题

1. 模板链 3'-TAC CGG TTA CGA ATC-5'。

(1) 编码链 (与模板互补、反平行)：5'-ATG GCC AAT GCT TAG-3'。

(2) mRNA (序列同编码链，T→U)：5'-AUG GCC AAU GCU UAG-3'。

(3) 密码子翻译：AUG=Met(起始)、GCC=Ala、AAU=Asn、GCU=Ala、UAG=终止。

多肽：Met-Ala-Asn-Ala (MANA)，共 4 个氨基酸。
