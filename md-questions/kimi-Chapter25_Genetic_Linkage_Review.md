Problem 1: Recombination Frequency and Map Distance (Drosophila)

In a classic experiment following Sturtevant's approach, a female Drosophila heterozygous for two X-linked genes was test-crossed:

• Body colour: wild-type (b+) vs. black body (b)

• Eye colour: wild-type (e+) vs. brown eyes (e)

The female had the genotype X^(b+e) / X^(be+) (cis arrangement). She was crossed to a male tester with genotype X^(be) / Y. The following offspring were observed:

| **Phenotype** | **Number** |
| --- | --- |
| Wild-type body, wild-type eyes | 412 |
| Black body, brown eyes | 398 |
| Wild-type body, brown eyes | 94 |
| Black body, wild-type eyes | 96 |

(a) Identify which offspring classes represent parental types and which represent recombinant types. Explain your reasoning. [3 marks]

(b) Calculate the recombination frequency (RF) between the body colour and eye colour loci. Show your working. [3 marks]

(c) Express the distance between these two genes in map units (mu) or centiMorgans (cM). [1 mark]

(d) If these two genes were assorting independently, what phenotypic ratio would you expect in the offspring? Explain why the observed data deviate from this expectation. [3 marks]

# Answer Key

The following are detailed solutions to the review problems.

## Problem 1: Answer

**(a)** Parental types: Wild-type body + wild-type eyes (412) and Black body + brown eyes (398). These are the most numerous classes because they inherit non-recombinant (parental) chromosome combinations. Recombinant types: Wild-type body + brown eyes (94) and Black body + wild-type eyes (96). These are less frequent because they result from crossing over between homologous chromosomes during meiosis.

**(b)** Total offspring = 412 + 398 + 94 + 96 = 1000. Recombinant offspring = 94 + 96 = 190. RF = (190 / 1000) x 100 = 19.0%.

**(c)** Map distance = 19.0 cM (or 19.0 map units). 1% recombination frequency = 1 cM.

**(d)** If assorting independently, the expected ratio would be 1:1:1:1 (25% each phenotype). The observed data deviate because both genes are on the same chromosome (X-linked) and are linked. The parental combinations occur much more frequently (81%) than recombinant combinations (19%), indicating linkage rather than independent assortment.

## Problem 2: Answer

**(a)** Total offspring = 6 + 62 + 58 + 5 = 131. If assorting independently, expected ratio = 1:1:1:1, so each class expected = 131 / 4 = 32.75. Chi-square calculation: (6-32.75)^2/32.75 + (62-32.75)^2/32.75 + (58-32.75)^2/32.75 + (5-32.75)^2/32.75 = 21.8 + 26.1 + 19.5 + 23.5 = 90.9. With df = 3, this far exceeds the critical value of 7.82 at p = 0.05. We reject independent assortment - the genes are linked.

**(b)** The F1 heterozygote is in the trans arrangement: N t / n T. Evidence: The most numerous offspring classes (62 normal coat/kinked tail and 58 flaky coat/normal tail) match the parental chromosome combinations from the original cross. The rare classes (6 and 5) are the recombinants. Parental chromosomes in F1: one chromosome carried N and t; the other carried n and T.

**(c)** Recombinants = 6 + 5 = 11. Total = 131. RF = (11 / 131) x 100 = 8.4%. Map distance = 8.4 cM.

**(d)** Genetic map: N ----8.4 cM---- T

## Problem 3: Answer

**(a)** Parental gametes: AB and ab. These are produced without crossing over and will be most frequent.

**(b)** Recombinant gametes: Ab and aB. These result from crossing over between the two loci.

**(c)** RF = 18%, so parental gametes each = (100% - 18%) / 2 = 41% of total. Recombinant gametes each = 18% / 2 = 9% of total. Expected offspring (500 total): Purple tall (AaBb from AB gamete) = 0.41 x 500 = 205. White dwarf (aabb from ab gamete) = 0.41 x 500 = 205. White tall (aaBb from aB gamete) = 0.09 x 500 = 45. Purple dwarf (Aabb from Ab gamete) = 0.09 x 500 = 45.

**(d)** In trans arrangement, the parental gametes would be Ab and aB (each 41%), and recombinant gametes would be AB and ab (each 9%). Expected: Purple tall = 45, White dwarf = 45, White tall = 205, Purple dwarf = 205. The phenotypic classes swap frequencies compared to the cis arrangement.

## Problem 4: Answer

**(a) & (b)**

| **Gene Pair** | **Independent or Linked?** | **Parental gametes (%)** |
| --- | --- | --- |
| A/a and B/b | Linked (15 cM) | AB (42.5%), ab (42.5%) |
| B/b and H/h | Linked (75 cM) | BH (12.5%), bh (12.5%) |
| D/d and G/g | Linked (25 cM) | DG (37.5%), dg (37.5%) |
| A/a and H/h | Linked (90 cM) | AH (5%), ah (5%) |

**(c)** Expected RF between A and H = (15 + 20 + 25 + 30) / 100 = 90%, which approaches 50% (the maximum observable RF). At 90 cM, the actual observed RF would be approximately 40-45% due to multiple crossovers. This illustrates that recombination frequency is a good proxy for distance only when genes are relatively close together (less than ~30 cM). At large distances, double crossovers become frequent and RF underestimates the true map distance.

## Problem 5: Answer

**(a)** First, identify parental (most numerous) and recombinant classes. Parental: red/normal/grey (380) and purple/vestigial/ebony (370). Between pr and vg: recombinants = 42 + 38 + 3 + 2 = 85. Total = 1000. RF(pr-vg) = 85/1000 x 100 = 8.5%. Between vg and e: recombinants = 85 + 80 + 3 + 2 = 170. RF(vg-e) = 170/1000 x 100 = 17.0%. Between pr and e: recombinants = 42 + 38 + 85 + 80 = 245. RF(pr-e) = 245/1000 x 100 = 24.5%.

**(b)** Gene order: pr --8.5 cM-- vg --17.0 cM-- e. Check: 8.5 + 17.0 = 25.5 cM, close to the observed 24.5% RF (small discrepancy due to double crossovers). Map: pr ----8.5 cM---- vg ---------17.0 cM--------- e

**(c)** Expected double crossovers = RF(pr-vg) x RF(vg-e) x total = 0.085 x 0.170 x 1000 = 14.5. Observed double crossovers = 3 + 2 = 5. Coefficient of coincidence (c) = observed DCO / expected DCO = 5 / 14.5 = 0.34. Interference (I) = 1 - c = 1 - 0.34 = 0.66. Interference of 0.66 means that a crossover in one region reduces the probability of a crossover in an adjacent region by 66%. This is positive interference, which is commonly observed in Drosophila.

**(d)** No, this conclusion is not justified. While the observed RF of 24.5% is less than 50%, it is significantly different from 50%. Genes at 25 cM are clearly linked, and a chi-square test would reject independent assortment. However, the RF does underestimate the true distance (25.5 cM) due to double crossovers.

**(e)** To distinguish cis from trans, examine which phenotype classes are most numerous in the test cross. In cis (pr+ vg+ e+ / pr vg e), the parental classes match the original homozygous parents. In trans (e.g., pr+ vg e+ / pr vg+ e), the most frequent offspring would have different phenotype combinations. The parental (most numerous) classes reveal the allele arrangement in the heterozygote.
